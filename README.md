# Landlourd

Search and review landlords in NYC. Filter by borough, rating, and sort by various criteria. Click any landlord card to see detailed info including contact details, portfolio stats, violations, and tenant reviews.

**Live site:** https://thisisworlddomination.github.io/landlourd/

## Features

- Search by name, address, borough, or zip code
- Filter by borough (Manhattan, Brooklyn, Queens, Bronx, Staten Island)
- Filter by minimum star rating
- Sort by rating, name, or number of properties
- Detailed modal view with contact info, portfolio stats, violation counts, and reviews
- Color-coded tags (green = positive, red = negative, blue = neutral)
- Responsive design for mobile and desktop

## Getting Started

### Run Locally

1. Clone the repo:
   ```sh
   git clone https://github.com/thisisworlddomination/landlourd.git
   cd landlourd
   ```

2. Open `index.html` in your browser:
   ```sh
   open index.html
   ```

   Or use any local server:
   ```sh
   python3 -m http.server 8000
   ```
   Then visit http://localhost:8000

### Deploy

The site is deployed via GitHub Pages from the `main` branch. Push to `main` to update the live site.

## Tech Stack

- Single-file static site (HTML, CSS, JavaScript)
- No dependencies or build step
- Dark theme with responsive layout
